# Endless Smart Contract Simulator

![Endless Logo](endless.png)

**Interactive Smart Contract Simulator for the Endless Ecosystem**  
Premium Web3 developer tool style prototype with neon-glow UI, AI-style insights, and interactive architecture map.

---

## 🌐 Demo

You can view the live demo here: https://endless-cont.lovable.app/

---

## 🎯 Features

- **Scenario Selector**: Token Transfer, NFT Mint, DAO Vote, Staking  
- **Simulation Flow Visualization**: User → Node → Execution → Storage  
- **Interactive Architecture Map** with clickable blocks and info popups  
- **AI Insights**: Step-by-step explanations (mock data)  
- **Wallet Connect**: Display mock onchain stats for connected wallet  
- **Neon-glow UI**: Dark theme, glassmorphism cards, animated particles  
- **Dark Mode / Neon Toggle**: Smooth color transitions  
- Fully responsive, mobile-friendly, smooth animations

---

## 🛠️ Technologies Used

- lovable ai
- Mock data for simulations  
- Glassmorphism + neon glow effects  

---

## 📌 Notes

This is a **prototype/demo** built to showcase a developer tool concept for the Endless ecosystem.  
All data is mock or random; there is **no backend required**.
